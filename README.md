## start point 💡

## Clone repo
```bash
git clone https://github.com/arainho/secure-git-workshop.git
cd secure-git-workshop || exit
```

## Define metadata for commits
```bash
git config user.name "Your Name"
git config user.email "your@mail.com"
```

## Done 👍 move to step 1
```bash
git checkout step1
```
